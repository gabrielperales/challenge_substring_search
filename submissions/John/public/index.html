      Adding Additional Tests/Engines

      How/Where to add more tests or Node Modules for this testing interface.

      Adding Additional Engines (Substring Search Files)

      Copy the new script into the base directory/custom_modules, and restart the server.
      The new script will be called in dynamically and will included in the next automated test run. Just make sure to refresh the browser too.

      Adding Additional Needles/Haystacks

      Copy the new text (.txt) file into either the base directory/testdata/needles || base directory/testdata/haystacks, Please make sure that the txt files are encoded utf-8, and that the needles are split by 1 space exactly. To see an example, open any of the files in the directory above.
